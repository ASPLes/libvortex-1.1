Vortex Library documentation

Vortex library documentation is composed by the API reference, manuals and tutorials. They are developed and maintained using doxygen tool.

If you find something not properly documented or missing or you think you could improve some section, contact us at the mailing list. It is really important to us to keep an useful, error free, and updated documentation.

Products using Vortex Library

Products, applications and network protocol implementations using Vortex Library:

  • Turbulence: General purpose, extensible BEEP application server.
  • jsVortex: A javascript BEEP implementation. jsVortex is using Vortex Library to validate its BEEP implementation.
  • PyVortex: Python bindings for Vortex Library 1.1. Official component included into base Vortex Library source code.
  • LuaVortex: Lua bindings for Vortex Library 1.1. Official component included into base Vortex Library source code.
  • BeepFp: BEEP services for Free Pascal (binding component written on top of Vortex Library 1.1).
  • CyBEX: Cybersecurity information exchange framework (X.1500)
  • Shaper: Creates xml documents by integrating data resulting from XML-RPC service execution.
  • Af-Arch: An Open Source N-Tier development framework.
