Vortex Library Professional services

Advanced Software Production Line, S.L. provides professional services for Vortex Library and BEEP based protocol development. They are defined from basic external tech support to BEEP profiles development.

What is included in the basic external tech support contract?

Commercial support highlights are:

  1. Priority support: which will provide you with valuable knowledge on how to use Vortex Library and how to fully integrate it with your environment.
  2. Direct and guaranteed tech support.
  3. Fast patch resolution.
  4. Not required to get support through the mailing list.
  5. Includes a commercial contract with confidential clauses.
  6. Product license assistance.

Why should I hire a Vortex Library support contract?

Vortex Library is an Open Source initiative created by ASPL. It is released under the LGPL license terms, allowing to develop Open Source as well as Commercial (closed source) project.

However, this doesn't ensure getting any tech support, patch resolution, etc, from ASPL. Currently, limited support may be found through the Vortex Library community mailing list.

To get priority and guarantee tech support to solve questions and fix bugs found while developing your project is only provided through a commercial contract.
