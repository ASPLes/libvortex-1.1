Vortex Library Professional services

Advanced Software Production Line, S.L. provides professional services for Vortex Library and BEEP based protocol development. They are defined from basic external tech support to BEEP profiles development.

What is included in the basic external tech support contract?

Commercial support highlights are:

  1. Priority support: which will provide you with valuable knowledge on how to use Vortex Library and how to fully integrate it with your environment.
  2. Direct and guaranteed tech support.
  3. Fast patch resolution.
  4. Produce binaries and installers for supported platforms.
  5. Not required to get support through the mailing list.
  6. Includes a commercial contract with confidentially clauses.
  7. Product license assistance.

Why acquire a Vortex Library support contract?

Vortex Library is an Open Source initiative created by ASPL. It is released under the LGPL license terms, allowing to develop Open Source as well as Commercial (closed source) projects.

However, this does not ensure getting any tech support, patch resolution, or similar services, from ASPL. Anyway, limited support may be found through the Vortex Library community mailing list.

To get priority and guarantee tech support to solve questions and fix found bugs while developing your project is only provided through a commercial contract.
